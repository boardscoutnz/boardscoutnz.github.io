About

This is a personal browser tool. The data shown here is built from a Trade Me board-game listings snapshot committed to the repository, joined against a pre-built BoardGameGeek metadata cache so you can sort and filter by BGG rank, rating, weight, player count, and playing time.

Tips

  • Multi-column sort: click a column to sort, hold Shift and click another column to add it as a lower-priority tiebreaker. The first column you click stays at priority 1; subsequent shift-clicks become 2, 3, etc. Numbered badges on each header show the active sort priority.
  • BGG link: click any matched game name in the BGG column to open its BGG page in a new tab.
  • Match status: the green check means a confident BGG match; orange means uncertain (top candidates were close); blank means no match could be made.
  • Data freshness: the green pill in the top bar shows when the listings snapshot was generated. It turns orange after a week and red after a month.
  • BGG columns hidden: the BGG-derived columns (BGG, Rank, Rating, Weight, Players, Time) are currently hidden because the BGG matching pipeline is still pending its API key. The disabled "Show BGG columns" toolbar button will be enabled once that work lands.
